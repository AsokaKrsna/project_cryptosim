Get Involved

Help shape CryptoSIM.

CryptoSIM is a community-driven project with a unique structure. Here is how you can get involved and help us build the ultimate visual cryptography tool.

⚙️ CORE APP // PROPRIETARY FREEWARE

Ideas & Bug Reports

The main CryptoSIM desktop application (the Rust backend, React Flow execution engine, and all 310 bricks) is proprietary freeware. To maintain intense focus and stability, we do not have a public GitHub repository for the core app — it cannot be starred or forked.

However, the app is 100% free to use forever, and its development is driven entirely by user feedback. If you find a bug or need a specific new brick, let us know!

🌐 WEBSITE & DOCS

Writing Guides

If you want to write a guide on how to build a specific cryptographic circuit, or if you spot a typo in the docs, we warmly welcome your contributions. Just send us an email with your suggestions or content!
